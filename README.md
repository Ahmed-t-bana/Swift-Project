# Swift-Project
# "University Management System Built in Swift"

# Project Description:
This project is a simple and organized university management system that simulates the management of students
faculty, courses, registrations, assignments, and submissions.

# Features:
- Student Management (with Major and GPA) 
- Faculty Management (with Specialization and Salary) 
- Course and Assignment Management 
- Registration and Submission Tracking 
- Use of Singleton for centralized system management

# Technologies Used...
- Programming Language: Swift 
- Model: Object-Oriented Programming (OOP) 
- Environment: Xcode - Swift Playground , https://www.programiz.com/swift/online-compiler/
